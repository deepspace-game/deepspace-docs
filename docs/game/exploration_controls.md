# Exploration Controls

|Default Key|Purpose|
|-----------|-------|
|Mouse Left|Turn Left|
|Mouse Right|Turn Right|
|Mouse Up|Fly up|
|Mouse Down|Fly down|
|Mouse Click|Interact with objects|
|W|Increase speed|
|A|Bank left|
|S|Decrease speed/reverse|
|D|Bank right|
|Q|Roll left|
|E|Roll right|
|Shift|Fast break|
|Space|Bank up|
|Ctrl|Bank down|
